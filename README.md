# Banking System in C++

A simple console-based Banking System developed in C++ using Object-Oriented Programming (OOP). This project demonstrates basic banking operations such as account creation, balance management, deposits, withdrawals, and transaction history.

## Features

- Create a new bank account
- View account details
- Deposit money
- Withdraw money
- Display transaction history
- Menu-driven interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Standard Template Library (STL)
  - vector
  - string

## Concepts Used

- Classes and Objects
- Constructors
- Functions
- Vectors
- Loops
- Switch Case
- Conditional Statements

## Project Structure

- `BankAccount` class
  - Stores account information
  - Handles deposits and withdrawals
  - Maintains transaction history
- `main()` function
  - Displays the menu
  - Handles user input
  - Performs banking operations

## How to Run

1. Compile the program:
   ```bash
   g++ banking_system.cpp -o banking_system
   ```

2. Run the program:
   ```bash
   ./banking_system
   ```
